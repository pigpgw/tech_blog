# AGENTS.md

## 프로젝트 규칙

- 사용자가 별도로 요청하지 않으면 `dev` 또는 작업 브랜치에서 작업한다.
- 프로젝트 문서는 한국어로 작성한다.
- 커밋 메시지는 `README.md`의 커밋 컨벤션을 따른다.
- 기록할 가치가 있는 AI 활용 사례는 `docs/04-ai-usage-log.md`에 남긴다.

## AI 공동 작성 기준

- Codex가 커밋되는 의미 있는 산출물을 직접 작성하거나 수정한 경우에만 공동 작성자 footer를 붙인다.
- 붙이는 경우:
  - Codex가 코드, 문서, 설정, 워크플로우, 스킬을 작성하거나 수정한 경우
  - Codex가 버그 원인을 진단하고 커밋되는 수정안을 제공한 경우
  - Codex가 커밋되는 문서 구조나 구현 계획을 재구성한 경우
- 붙이지 않는 경우:
  - 사용자가 실제 작업을 했고 Codex는 질문 답변, 가벼운 검토, 표현 확인만 한 경우
  - 단순 포맷 변경, 간단한 질문, 낮은 가치의 보일러플레이트 생성인 경우

footer:

```txt
Co-authored-by: Codex <codex@openai.com>
```

- body가 있으면 body 아래에 둔다.
- body가 없으면 subject 아래에 한 줄을 띄우고 둔다.
